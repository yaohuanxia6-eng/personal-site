Section 01

产品概述

1.1 产品名称

色彩炼金（英文：Color Alchemy）

1.2 产品定位

一款面向儿童及色彩爱好者的网页端颜色混合实验小游戏。用户通过将不同颜色"倒入"虚拟试管，观察颜色混合后的变化，探索色彩搭配的乐趣，并可将自己喜欢的混合色收藏命名。

1.3 产品类型

轻量级 Web 小游戏 / 教育互动工具

1.4 一句话描述

"像炼金术师一样调配颜色——选颜色、倒试管、看变化、存收藏。"

1.5 核心价值

  • 寓教于乐：让用户通过动手实验直观理解色彩混合原理（RGB 加法混色）
  • 零门槛：无需下载安装，浏览器打开即玩，操作极简
  • 创造感：用户可以创造属于自己的独特颜色，命名并收藏

8.4 色卡收藏流程

  1. 用户点击"色卡收藏"按钮
  2. 系统查找第一根有内容的试管（从试管 1 到试管 4 依序）
  3. 所有试管为空 → alert("请先在试管中倒入颜色")
  4. 找到 → 记录该试管索引和混合色值
  5. 弹出命名弹窗，默认值"新颜色"，光标自动聚焦
  6. 用户输入 → 点击"确认"或按 Enter
  7. 名称超过 4 个字 → alert("颜色名称最多4个字")，不关闭弹窗
  8. 名称为空 → 关闭弹窗，不保存
  9. 收藏已满 8 个 → alert("最多只能收藏8个颜色")，关闭
  10. 否则 → 添加收藏 → 清空试管 → 更新 UI → 关闭弹窗
Section 09

颜色混合算法说明

9.1 算法原理

采用 RGB 通道均值混合算法。将试管内所有颜色的 R、G、B 三通道分别求和后取平均值。

9.2 计算公式

混合色.R = floor( (色1.R + 色2.R + … + 色N.R) / N )
混合色.G = floor( (色1.G + 色2.G + … + 色N.G) / N )
混合色.B = floor( (色1.B + 色2.B + … + 色N.B) / N )

其中 N = 试管内颜色的层数（1~6）

9.3 混合示例

输入颜色RGB混合结果
红 + 蓝(255+0)/2=127(0+0)/2=0(0+255)/2=127#7F007F 暗紫
红 + 黄(255+255)/2=255(0+255)/2=127(0+0)/2=0#FF7F00 偏橙
红 + 绿 + 蓝(255+0+0)/3=85(0+255+0)/3=85(0+0+255)/3=85#555555 深灰

9.4 颜色格式转换

  • HEX → RGB：正则解析 #RRGGBB 为三个十六进制值，转 parseInt
  • RGB → HEX：位运算 (1 << 24) + (r << 16) + (g << 8) + b 转 16 进制取后 6 位
已知局限RGB 均值混合并非真实颜料混色（减法混色）。如"红+蓝"在现实中会得到紫色，但 RGB 均值得到的是暗紫。未来可考虑引入 CMYK 减法混色或更精确的色彩模型。
